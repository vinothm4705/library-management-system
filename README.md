# Library Management System

A web-based Library Management System built with PHP and MySQL.

## Features

- Secure login, signup, and logout with session management
- Manage Books — Add, View, Edit, and Delete records
- Manage Students — Add, View, Edit, and Delete records
- MySQL database integration for reliable data storage
- Clean and responsive UI built with HTML and CSS

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- WAMP Server (local development)

## Project Setup

1. Install WAMP or XAMPP on your computer
2. Clone this repository into your `www` folder
3. Import the database:
   - Open phpMyAdmin
   - Create a new database named `library`
   - Import the SQL file from the project folder
4. Rename `config.sample.php` to `config.php`
5. Update database credentials in `config.php`
6. Open browser and go to `http://localhost/lbms`
